Use HTML to define Progressbar

    <brick-progressbar value="60"></brick-progressbar>

Striped

    <brick-progressbar value="15" striped></brick-progressbar>

Striped and Active

    <brick-progressbar value="20" striped active></brick-progressbar>

Set Progress from JavaScript

    <brick-progressbar id="p1" value="20" striped active></brick-progressbar>
    <button onclick="document.getElementById('p1').value=40;">Click to Set 40%</button>

Min/Max

    <brick-progressbar min="100" max="900" id="p2" value="300" striped active></brick-progressbar>
    <button onclick="document.getElementById('p2').value=800;">Click to Set 800</button>

Min/Max with Showing Actual Values

    <brick-progressbar min="100" max="900" showPercentage="false" id="p3" value="300" striped active></brick-progressbar>
    <button onclick="document.getElementById('p3').value=800;">Click to Set 800</button>

Toggle between Percentage and Actual Values

    <brick-progressbar min="100" max="900" showPercentage="false" id="p4" value="300" striped active></brick-progressbar>
    <button onclick="document.getElementById('p4').showPercentage=!document.getElementById('p4').showPercentage;">Toggle</button>
