Use HTML to define Progressbar

        <brick-progressbar value="10"></brick-progressbar>
    
        <brick-progressbar value="15" striped></brick-progressbar>
    
        <brick-progressbar value="20" striped active></brick-progressbar>

Set Progress from JavaScript

        <brick-progressbar id="p1" value="20" striped active></brick-progressbar>
        <button onclick="document.getElementById('p1').value=40;">Click to Set 40%</button>
